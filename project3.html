Doctor E-Booking Portal

Full Stack Developer

Designed & built an online Doctor booking portal using MERN Stack where patients can able to
get their e-appointment & confirmation completely by online system. This would reduce very long queues
in appointment desks in the hospitals and support the patients for treatment on thier preferred Date.
Role based authentication is implemented by using JWT. Seperate UI was created where user can register, signin,
signup and reset the password by using forgotpassword. Nodemailer is used to send the e-appointment confirmation
with all details once user submitted the application. A scalable and maintainable codebase for future enhancements and improvements.
